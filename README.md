# Pet Adoption Management System

## Overview
The Pet Adoption Management System is designed to facilitate the adoption process of pets by connecting potential pet owners with shelters and fostering organizations. This system streamlines the process of managing pet information, applications, and adopter communications.

## Features
- **User Registration and Login:** Users can create an account and log in to manage their profiles and applications.
- **Profile Management:** Users can update their profiles, including personal details and preferences for adoptable pets.
- **Pet Listings:** View detailed information about available pets, including images, descriptions, and adoption requirements.
- **Adoption Applications:** Users can submit applications for pet adoption directly through the system.
- **Admin Dashboard:** Admins can manage pet profiles, view applications, and communicate with users.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript (React.js)
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/gunjan2001/pet-adoption-management-system.git
   ```
2. Navigate into the directory:
   ```bash
   cd pet-adoption-management-system
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables as required.
5. Start the server:
   ```bash
   npm start
   ```

## Usage
- Open your browser and go to `http://localhost:3000` to access the application.
